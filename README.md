# nxt-connect

nxt-connect is a REST API client that wrapped NXT Client's Web API for simple access, it has a built-in blockchain peers indexer to ensure your requests will never fail if any peer(client) is offline. We officialy support [NXT Wallet](https://demo.nxtplatform.org/index.html) from the official [NXT Platform](https://nxtplatform.org/),  [WCG Wallet](https://wcgacc.com/) from three blockchain cryptocurrency exchange([EUNEX](https://eunex.co/), [MBAEX](https://mbaex.com/#/index), [WCEEX](https://www.wceex.com/)), and any existing blockchain wallets uses [NXT Client](https://nxtplatform.org/download/).
